## 프로젝트 개요
JPA/Hibernate 이론을 공부하며 간단한 CRUD 실습으로 게시판 만들기

## 게시판 기능
* CRUD
* 파일 업로드
* 페이징
* 검색 

## 사용 기술
### 백엔드
#### 주요 프레임워크 / 라이브러리
- Java 11
- SpringBoot 2.6.8
- JPA(Spring Data JPA)

#### Build Tool
- Maven

#### DataBase
- MySQL 8.0.29

### 프론트 엔드
